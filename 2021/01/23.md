# 4Sum II
***2021/01/23***
## <span class="orange">Medium</span>

```python=
class Solution:
    def fourSumCount(self, A: List[int], B: List[int], C: List[int], D: List[int]) -> int:
        dict1 = {}
        sum1 = 0
        #分成2部分，用dict去存A+B，每找到一組就把當前key的值+1，再用-(C+D)去找對應的解
        for i in A :
            for j in B:
                key = i + j
                if key in dict1 :
                    dict1[key] += 1
                else :
                    dict1[key] = 1
                
        for i in C :
            for j in D:
                key = -(i + j)
                if key in dict1:
                    sum1 += dict1[key]
        return(sum1)
```
